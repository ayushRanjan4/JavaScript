# JavaScript Learning Repository

A hands-on JavaScript practice workspace with topic-wise exercises and mini projects.

This repository is organized to move from fundamentals to practical browser projects. It is useful for beginners who want to practice concepts in small, focused files.

## What Is Inside

- Core JavaScript basics (variables, data types, conversions, strings, arrays, objects, functions, scopes, arrow functions, IIFE)
- Control flow and iteration practice
- DOM-focused examples
- Mini projects built with HTML, CSS, and JavaScript
- One Node.js console example using prompt input

## Folder Structure

```text
basic/
control_flow/
DOM/
Iterations/
project/
  BMI Calculator/
  clock/
  Color Scheme Switcher/
  Guess a number/
package.json
```

## Learning Path (Recommended)

1. Start with `basic/` for fundamentals.
2. Move to `control_flow/` for decision making logic.
3. Practice loops and array methods in `Iterations/`.
4. Learn browser interaction in `DOM/`.
5. Build confidence by running projects in `project/`.

## Mini Projects

### 1) BMI Calculator
- Path: `project/BMI Calculator/`
- Focus: Form handling, input validation, DOM updates, condition-based output.

### 2) Color Scheme Switcher
- Path: `project/Color Scheme Switcher/`
- Focus: Event listeners, dynamic styling, user interaction.

### 3) Guess a Number
- Path: `project/Guess a number/`
- Focus: Random number logic, state tracking, attempts management, feedback messages.

### 4) Clock
- Path: `project/clock/`
- Focus: Date object usage and real-time UI updates.

## Prerequisites

- Node.js (only needed for console-based file `Iterations/08.js`)
- Any modern browser (Chrome, Edge, Firefox)

## Setup and Run

### Clone and Install

```bash
git clone https://github.com/ayushRanjan4/JavaScript.git
cd JavaScript
npm install
```

### Run Node Console Example

```bash
node Iterations/08.js
```

### Run Browser Projects

Open any `index.html` file directly in a browser, for example:

- `project/Guess a number/index.html`
- `project/BMI Calculator/index.html`
- `project/Color Scheme Switcher/index.html`
- `project/clock/index.html`

Tip: You can also use the VS Code Live Server extension for faster iteration.

## Dependency

- `prompt-sync`: used in `Iterations/08.js` for terminal input.
